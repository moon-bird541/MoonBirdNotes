# MoonBirdNotes

## 项目简介

MoonBirdNotes 是一个面向个人使用的在线 Markdown 知识库系统。
它主要用于统一管理个人文章、学习笔记和日记内容，帮助用户完成笔记的上传、整理、查看、编辑、删除与长期保存。

当前系统以前后端分离方式实现，前期仅支持管理员本人登录使用，不开放用户注册，所有内容默认私密保存。
后续可以继续扩展搜索、标签筛选、智能问答以及多用户能力。

## 技术栈

- 后端：Django
- 接口：Django REST Framework
- 认证：JWT
- 数据库：MySQL
- 前端：Vue 3
- UI 组件：Element Plus
- 构建工具：Vite

## 当前已实现功能

- 管理员登录
- 上传 Markdown 笔记
- 自动保存原始 Markdown 文件
- 自动渲染 Markdown 为 HTML
- 标签必填，且每篇笔记最多两个标签
- 支持选择已有标签或创建新标签
- 笔记列表分页展示
- 列表展示标题、摘要、字数、标签、时间信息
- 笔记详情页展示完整内容
- 在线编辑笔记标题、正文和标签
- 编辑历史版本记录与版本回退
- 无实际修改时不生成新的历史版本
- 笔记软删除
- 回收站恢复笔记
- 回收站彻底删除笔记
- 回收站 30 天自动清理
- 左侧功能栏收起与展开

## 历史版本规则

在线编辑笔记时，系统会在真正发生修改之前自动保存修改前的内容为历史版本。
每篇笔记最多保留最近 5 个历史版本。
如果用户保存时标题、正文和标签都没有变化，则不会生成新的历史版本，也不会刷新更新时间。
